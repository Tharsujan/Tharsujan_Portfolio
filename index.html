Recent Projects

Group Project

Agriculture Consultancy App

Our group project, the Agriculture Consultancy App, is a mobile application developed using Flutter and Dart, with Firebase for database connection. This app provides consultancy services to farmers and individuals in the agriculture sector. Users can seek expert advice on various agricultural practices, access educational resources, and engage with a community of fellow farmers. The app aims to empower users with the knowledge and support needed to improve agricultural productivity and sustainability.

Individual Project

Travel and tourist Website

My individual project, the Travel and Tourist Website, is a dynamic web platform developed using HTML, CSS, JavaScript, PHP, Bootstrap, and MySQL for database connectivity. This website showcases various travel packages and destinations for users to explore. Visitors can view detailed descriptions of each package, including itinerary, accommodation options, and pricing. The website allows users to select and book their desired travel package directly through the platform. With a user-friendly interface and seamless booking process, the website aims to provide travelers with convenient access to exciting travel experiences.

Individual Project

Responsive Travel and tourist Website

My individual project, the Responsive Travel and Tourist Website, is a frontend-only web application built using React.js. This website features a collection of destinations and travel packages for users to explore. Visitors can browse through various destinations and view details of each package, including destination highlights, activities, and accommodations. Additionally, travelers have the option to contact me directly for inquiries or bookings. With a focus on responsiveness and user experience, the website offers a seamless browsing experience across different devices, catering to travelers seeking exciting destinations and memorable experiences.

Group Project

Online University notice board system

Our group project is an online notice board system for universities. Built with HTML, CSS, JavaScript, PHP, and MySQL, it allows students to easily access and manage notices for courses, events, and announcements. They can pin or save notices of interest. Staff members can also publish, view, and edit notices. The system streamlines communication within the university community, providing a user-friendly platform for efficient notice management.
